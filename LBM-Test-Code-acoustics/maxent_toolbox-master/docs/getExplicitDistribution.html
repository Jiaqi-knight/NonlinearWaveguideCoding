maxent.getExplicitDistribution

Description

Returns an explicit representation of a probability distribution as a vector of probabilities. This function will return an error for large (default ≥ 30) distributions since they typically cannot fit in memory.

Usage

probabilities = maxent.getExplicitDistribution(model)

Arguments

Mandatory arguments

  • model - Probabilistic model as returned by the trainModel function.

Output

  • probabilities - vector of probabilities for all system states starting from 00000, 00001.... up to 11111.
